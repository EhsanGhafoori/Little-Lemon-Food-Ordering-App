# User Stories (9)

## 1) Sign Up
As a new user, I want to create an account with my details so that I can start using the app.

## 2) Login
As a registered user, I want to log in with email and password so that I can access my account.

## 3) Signup/Login Errors
As a user, I want to see clear error messages when authentication fails so that I can correct my input.

## 4) Home Screen
As a user, I want to see a home screen with app branding and main content so that I can navigate quickly.

## 5) Detail Screen
As a user, I want to open a detail page from the main screen so that I can view full item information.

## 6) Local Storage Persistence
As a user, I want important app data saved locally so that my session/data is available after reopening the app.

## 7) External API Integration
As a user, I want the app to fetch data from an external API so that displayed content is dynamic and up to date.

## 8) Settings Menu and Screen
As a user, I want to open the settings menu and settings page so that I can configure preferences.

## 9) Notifications
As a user, I want to configure notifications and receive test alerts so that I stay informed.
# TaskHub - User Stories

## 1. Login & Registration Page
* **Account registration/login:** Sebagai pengguna, saya ingin mendaftar dengan detail saya atau masuk menggunakan kredensial yang sudah ada agar saya bisa mengakses akun saya.
* **Error feedback:** Sebagai pengguna, saya ingin menerima pesan *error* ketika saya memasukkan username atau password yang salah agar saya bisa memperbaikinya.

## 2. Home Page
* **Welcome message:** Sebagai pengguna, saya ingin melihat pesan selamat datang atau kutipan motivasi di halaman utama agar saya merasa terhubung dengan aplikasi.
* **Display progression:** Sebagai pengguna, saya ingin melihat ringkasan progres tugas saya di halaman utama agar saya tahu sejauh mana pencapaian saya hari ini.
* **View completed tasks:** Sebagai pengguna, saya ingin bisa melihat tugas-tugas yang sudah diselesaikan langsung dari halaman utama.

## 3. Navigation & Menu
* **Access menu options:** Sebagai pengguna, saya ingin mengakses menu navigasi agar saya bisa berpindah antar bagian aplikasi dengan mulus.
* **Navigate to profile:** Sebagai pengguna, saya ingin menekan tombol di menu untuk pergi ke halaman profil saya.
* **Go back to home:** Sebagai pengguna, saya ingin memiliki cara mudah untuk kembali ke halaman utama dari menu mana pun.

## 4. Tasks Management
* **Add/delete/move tasks:** Sebagai pengguna, saya ingin bisa menambahkan tugas baru, menghapus tugas lama, dan memindahkan status tugas agar manajemen kerja saya terorganisir.
* **Personalize tasks:** Sebagai pengguna, saya ingin mempersonalisasi tugas saya (misalnya memberi label atau warna) agar lebih mudah dikenali.

## 5. Detailed Screen
* **Task details & instructions:** Sebagai pengguna, saya ingin melihat deskripsi singkat dan instruksi detail dari sebuah tugas saat saya mengkliknya agar saya tahu persis apa yang harus dikerjakan.

## 6. Profile Page
* **View/edit personal info:** Sebagai pengguna, saya ingin melihat dan memperbarui detail registrasi dan informasi pribadi saya di halaman profil.

## 7. Reports Page
* **View daily/weekly tasks:** Sebagai pengguna, saya ingin melihat laporan tugas harian atau mingguan agar saya bisa mengevaluasi produktivitas saya.
* **Completed/pending tasks:** Sebagai pengguna, saya ingin melihat perbandingan jelas antara tugas yang selesai dan yang masih tertunda di halaman laporan.

## 8. Notifications & Reminders
* **Enable/disable notifications:** Sebagai pengguna, saya ingin bisa menyalakan atau mematikan notifikasi agar saya tidak terganggu saat sedang fokus.
* **Add reminder:** Sebagai pengguna, saya ingin menambahkan pengingat pada tugas spesifik agar saya tidak melewatkan tenggat waktu.
* **Set notification calendar:** Sebagai pengguna, saya ingin mengatur kalender notifikasi agar pengingat sesuai dengan jadwal harian saya.

## 9. Sharing of Tasks
* **Share via external apps:** Sebagai pengguna, saya ingin bisa membagikan atau merekomendasikan tugas ke teman/keluarga melalui media sosial, email, atau aplikasi pesan instan.

## 10. Settings & Customization
* **Change theme:** Sebagai pengguna, saya ingin mengubah latar belakang aplikasi dari *light mode* ke *dark mode* (atau tema lain) agar nyaman di mata.

## 11. Security & Data
* **Log out securely:** Sebagai pengguna, saya ingin bisa *log out* dengan aman, di mana aplikasi menghapus sesi data saya dan mengembalikan saya ke halaman login.
* **Persistent Data:** Sebagai pengguna, saya ingin sesi saya dan preferensi pengaturan (seperti *dark mode*) tetap tersimpan secara lokal saat aplikasi ditutup.
* **External API Integration:** Sebagai pengguna, saya ingin aplikasi dapat menarik data dari luar (misalnya API kalender atau cuaca) untuk mendukung perencanaan tugas saya.
